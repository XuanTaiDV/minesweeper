# README

A dangerous area make we feel alive!

## Setup

```
$ git clone git@github.com:XuanTaiDV/minesweeper.git
$ cd minesweeper
$ bundle install
$ rake db:create
$ rake db:migrate
$ rails webpacker:install
$ rails webpacker:compile
$ rails s
```

Access to http://localhost:3000/ and create a board games!
